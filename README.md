# FazendaJS

Sistema de gestão de matrizes bovinas.

## Estrutura

- `index.html`: estrutura principal.
- `assets/css/style.css`: layout estilo aplicativo.
- `assets/js/app.js`: lógica original preservada.
- `manifest.json` e `sw.js`: PWA instalável e cache offline.
- `database/supabase.sql`: base futura para Supabase.

## Hospedagem GitHub Pages

Publique a branch `main` em **Settings > Pages > Deploy from a branch > main / root**.

## Observação

A versão atual preserva o uso de LocalStorage. A integração Supabase está preparada em arquivos de exemplo, mas ainda não substitui a persistência local.
